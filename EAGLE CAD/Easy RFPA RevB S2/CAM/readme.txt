Readme for Easy_RFPA_RevB_S2 CAM files
Mike Twieg, April 30, 2022

Material: FR4 1/16" finished thickness
Number of layers: 2
All copper layers 1oz
Surface finish: HASL
Soldermask (green) on both sides
Silkscreen (white) on both sides
Ignore silkscreen features which extend beyond the board dimensions, and which overlap with soldermask openings.

Panel description:
Board dimensions: 6.000" x 6.000", indicated on the outline layer
No array, boards to be fabbed 1-up
